# Basic-Discord-Bot-Starter
Just has the basics for a discord bot that is all. This is my first repository :) 
- Must make sure you have Node.js installed.
- Ensure all files are in the same folder.
- You will need to install the packages yourself, here is how to do it ↓
# 1
Open a terminal in your environment
- Type the following 'npm init -y'
- You should get a package.json file
# 2
Type the following 'npm install discord.js'
- You should get a package-lock.json
# 3 
After you have pasted your token in the config.json here use the terminal to turn it on.
- Type the following 'node index.js'
- If you get 'Error: Cannot find module' it most likely means the files are not in the same folder.
